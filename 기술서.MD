## 1. 프로젝트 개요 - **프로젝트명**: 부울경 소상공인 AI 마케팅 매니저 - **목표**: 부울경 소상공인(카페·식당·동네 가게 등)이 **사진 + 간단한 정보만 입력하면**, 인스타 / 스토리 / 지도 리뷰 / 문자 홍보 문구를 **자동 생성**해주는 웹 서비스. - **핵심 가치** - “글 잘 못 써도, 바빠도, 매일 홍보가 가능하게” - 일반 챗봇이 아니라 **부산 데이터·부산 감성**이 들어간 **로컬 특화 마케팅 도구**. --- ## 2. 기술 스택 ### 2.1 Frontend - **프레임워크**: React - **번들러/빌드**: Vite - **언어**: TypeScript (선호) 또는 JavaScript - **UI 라이브러리** - 기본: Tailwind CSS (또는 styled-components / emotion 택1) - 아이콘: react-icons 등 - **상태 관리** - 규모 상: React Query + React Context (또는 최소한의 useState) - **HTTP 통신** - fetch API 또는 axios ### 2.2 Backend (간단 버전 가정) > 해커톤 시간 제약상 **얇은 BFF(Backend For Frontend)** 느낌으로 최소 구현 - **언어/런타임**: Node.js - **프레임워크**: Express (혹은 FastAPI 등 팀 편한 것) - **기능 역할** - LLM API(Solar) 프록시 - Big-데이터웨이브 / 축제 데이터 등 정제 후 프롬프트에 주입 - 간단한 로그/통계 저장 (선택) ### 2.3 AI/데이터 - **LLM**: 업스테이지 Solar API (한국어/부산 사투리 지원) - **데이터** - 부산 축제 / 유동인구 / 관광 데이터 (Big-데이터웨이브 → CSV/JSON) - 날씨/요일/시간대와 묶어서 “상황 컨텍스트”로 사용 --- ## 3. 전체 시스템 아키텍처
text

가게 이름

오늘/이번 주 이벤트 간단 텍스트

 세련된 인스타 감성 (서울말)

채널 선택 (checkbox)

 인스타 스토리용

 지도 리뷰용

 문자/알림톡용

사진 업로드는 최소 Mock 또는 file input만 두고, 실제 업로드는 생략 가능 (시간에 따라 결정).
프롬프트에서는 “사진 설명” 텍스트 필드로 대체 가능.

4.1.2 출력 항목
인스타 피드 문구 + 해시태그 리스트

인스타 스토리 짧은 문구

네이버/카카오 지도 리뷰 문구

문자/알림톡용 1~2줄 문구

4.1.3 UX 플로우
사용자 입력 → “AI에게 맡기기” 버튼 클릭

로딩 스피너 표시

채널별 탭/아코디언 형태로 결과 표시

각 블록 옆에 “복사하기” 버튼 (클립보드 copy)

4.2 축제·상권 이벤트 연동 추천
4.2.1 컨셉
현재 날짜 기준으로 부산 이벤트/축제 데이터를 읽어,

해당 상권(예: 광안리, 서면)에 관련 이벤트가 있으면

“오늘은 ○○축제라서 이런 콘셉트 문구를 추천합니다” 식 문구 생성

4.2.2 처리 방식 (해커톤 최적화 버전)
데이터는 Backend에서 미리 로딩한 JSON/CSV 사용

오늘 날짜와 사용자가 선택한 “상권/동 이름” 기준으로 간단 필터

LLM 프롬프트에 Context로 주입:

오늘 광안리에는 불꽃축제가 있고, 방문객이 많을 것으로 예상된다. 이를 고려해 '축제 전 들르기 좋은 카페' 컨셉으로 홍보문구를 작성해라.

4.2.3 Frontend UI
“오늘 우리 동네 이슈 불러오기” 버튼

버튼 클릭 시:

백엔드에서 contextSummary 텍스트 반환 (예: “오늘 광안리 불꽃축제, 서면 플리마켓 진행 중”)

이 컨텍스트를 바로 프롬프트에 태워서 글 생성

4.3 마케팅 캘린더 & 리마인드 (MVP 축소 버전)
완전한 캘린더 구현 대신, “오늘/주말/급여일” 정도만 선택할 수 있는 버튼형으로 축소.

4.3.1 기능
“언제 쓸 글인가요?” 선택:

 오늘 평일 낮

 이번 주말

 월급날 전후

 지역 축제 당일

선택값은 LLM 프롬프트에 컨텍스트로만 사용
(실제 알림/푸시 구현은 시간 되면 추가)

4.3.2 프롬프트 반영 예
“이번 주말에 손님이 몰리는 시간을 노리고, 가족/연인 단위 손님에게 어필하고 싶다.”

“급여일 전이라 가격에 민감한 고객을 대상으로 가성비를 강조하라.”

4.4 Tone & Manner (서울말 vs 부산 사투리)
4.4.1 UX
라디오 버튼:

◉ 세련된 인스타 감성 (서울말)

○ 친근한 부산 사투리 (단골 문자 스타일)

4.4.2 프롬프트 예시
서울말:

“세련되지만 과하지 않게, 20~30대 인스타 감성에 맞춰 문장을 작성해라.”

부산 사투리:

“부산 사투리를 활용해서, 단골 손님에게 말하듯 친근하게 작성해라.
예: ‘오늘 날씨 억수로 춥네예. 따뜻한 라떼 한 잔 하이소.’ 같은 톤.”
